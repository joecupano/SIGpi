Welcome to SIGpi-installer

Given a Raspberry Pi4 with 4GB RAM, a 32GB microSD with Raspberry Pi OS Full (32-bit) installed, 
this script installs drivers and applications for signals intelligence (SIGINT) and Amateur Radio use.

Installation will continue with the following steps:

- Choose Devices
- Select GNUradio version
- Select Digital Decoders
- Choose SDR applications
- Choose Amateur Radio Library and Applications
- Choose Packet Radio Applications
- Choose Additional Tools

This installation can take four hours to install. Example compile/build times include:
- GNuradio 3.8 (~50 minutes)
- SDRangel  (~100 minutes)
- Hamlib 4.3 (~45 minutes)
- Fldigi Suite (~30 minutes)
- QSSTV 9.5.8 (~30 minutes)
